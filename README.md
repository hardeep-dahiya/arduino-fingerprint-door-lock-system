# Smart Biometric Door Lock System

## Overview
This project is an Arduino UNO based smart door lock system that uses an R307 optical fingerprint sensor for biometric authentication. The system unlocks the door automatically using a servo motor when an authorized fingerprint is detected.

## Features
- Fingerprint Based Authentication
- Automatic Door Unlocking
- Secure Access Control
- Servo Motor Lock Mechanism
- User Enrollment and Verification

## Components Used
- Arduino UNO
- R307 Optical Fingerprint Sensor
- Servo Motor
- Jumper Wires
- Power Supply

## Working Principle
The fingerprint sensor captures and verifies the user's fingerprint. If the fingerprint matches the stored data, Arduino sends a signal to the servo motor to rotate and unlock the door. After a delay, the servo motor returns to the lock position.
